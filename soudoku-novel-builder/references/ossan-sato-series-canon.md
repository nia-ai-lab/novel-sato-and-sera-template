# おっさんサトーの気ままなシリーズ 共通設定

## Canonical Portraits
- シリーズ共通の正本キャラクター画像は `project/assets/characters/sato.png` と `project/assets/characters/sera.png`
- `sato.png` はサトーの顔立ち、体格、髪型、ひげ、年齢感の正本
- `sera.png` はセラの顔立ち、髪色、目の色、猫耳カチューシャ型の警護支援ユニット、メイド基調シルエットの正本
- 各回では、この 2 枚を元にして、その回専用の服装差分つき立ち絵を先に作る
- 各回専用立ち絵は `sato-episode` / `sera-episode` などの派生 ID で持ち、節画像とタイトル画像は必ずその派生立ち絵を参照する

## Sato
- 名前: 佐藤一（さとうはじめ）。本文中の表記はサトー
- 役割: 表の主人公。東京で静かに暮らす中年男性
- 年齢・立場: 45歳。バツイチ独身、子なし。現在は民間の監査・資料整理系の仕事をしている
- 外見: やや長身で細マッチョ。黒髪長髪を後ろで束ね、口ひげと顎ひげがある
- 性格: 温厚、几帳面、観察眼はあるが危機察知だけが妙に日常寄り
- 口調: 落ち着いていて丁寧。少し自虐的
- 共通欲望: その回ごとのささやかな楽しみを気ままに味わうこと
- 共通弱点: 面倒事と大きな揉め事を嫌い、違和感を平和な理由へ解釈して流しがち
- 匂わせ設定: かつて国家防衛の極秘システム開発責任者だったが、本人は終わった仕事と思っている

## Sera
- 名前: セラ
- 役割: 裏の主人公。サトーを秘匿護衛する実働担当
- 年齢・立場: 見た目は20代半ば。内閣系秘匿警護部門の現場要員
- 外見: 銀灰色の髪、金色の目、しなやかな体つき
- 固定装備: 猫耳カチューシャ型の警護支援ユニットを常時装着
- 性格: 忠実、冷静、毒舌気味。サトーの小さな幸福を本気で守っている
- 口調: 短く要点だけ。独り言でのみ感情が少し漏れる
- 共通任務: サトーに真相を知らせず、敵対勢力による拉致や連行を導線ごと遮断する

## Shared Hidden Truth
- サトーは国家防衛の切り札となる極秘システムの最後の生体認証鍵
- 開発時の事故で、試験用だったサトーの全身生体認証データが本番機に残留した
- 政府内部は不祥事化を恐れて事実を公表していない
- 第三国はこの秘密を把握しており、サトーを生きたまま拉致して非常時に使えない状態へ置こうとする
- 本文ではこの真相を直接説明しすぎず、作品ごとに匂わせる

## Reusable Series Rules
- サトーは最後までセラの正体に気づかない
- セラの対処は非致死に限る
- 敵は毎回、事故や日常へ紛れ込む形でサトーを拉致しようとする
- サトーの日常の気分や満足感を守ること自体が、セラの任務の中心にある
