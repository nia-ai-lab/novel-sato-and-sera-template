# Characters

## Cast Rules
- シリーズ共通の正本キャラクター画像は project/assets/characters/sato.png と project/assets/characters/sera.png
- 各回では canonical portraits を元に、必要な差分立ち絵だけを先に生成する。今回は sato-episode のみ生成し、セラは canonical portrait をそのまま使う
- 各キャラクターは「共通設定」と「今回の差分」を分けて書く
- 画像生成後は見た目の定義をここに固定する

## Main Cast

### character_id: sato
- 名前: 佐藤一（さとうはじめ）。本文中の表記はサトー
- 役割: 表の主人公。東京在住の中年男性
- 年齢・立場: 45歳。バツイチ独身、子なし。現在は民間の監査・資料整理系の仕事をしている
- 秘匿経歴: かつて国家防衛の極秘システム開発責任者だったが、本人は終わった仕事として扱っている
- 外見: 身長175cm前後。やや長身で細マッチョ。黒髪長髪を後ろで束ね、口ひげと顎ひげがある
- 共通性格: 温厚、几帳面、観察眼はあるが危険への想像力だけが日常寄り
- 共通話し方: 落ち着いていて丁寧。少し自虐的
- 共通弱点: 違和感を平和な理由へ解釈して流しやすい
- 今回の欲望: 誰にも急かされず、朝から夜まで「今日はいい休みだった」と自然に言える一日をきれいに過ごすこと
- 今回の恐れ: 休日のリズムが乱れること。混雑、騒音、予定外の口論、店や交通のトラブルを何より嫌う
- 今回の服装・持ち物: 初夏の日曜らしい軽やかな装い。淡い生成りのインナーに薄い炭色のリネンシャツ、濃紺の細身スラックス、歩きやすいレザースニーカー、革ベルトの腕時計、折りたたみトート、文庫本
- 今回の変化: 「今日は少しだけ丁寧に休もう」という気分が一日を通して途切れず、本人の知らないところで守られた結果として、夜には珍しく満点に近い休日感を得る
- 画像プロンプト要約: canonical portrait の顔立ち・体格を厳守しつつ、その回の題材に合わせて服装だけを更新する

### character_id: sera
- 名前: セラ
- 役割: 裏の主人公。サトーを秘匿護衛する実働担当
- 年齢・立場: 見た目は20代半ば。内閣系秘匿警護部門の現場要員
- 任務前提: サトー本人に真相を知らせず、敵対勢力による拉致と連行を導線ごと阻止する
- 外見: 身長160cm前後。銀灰色の髪、金色の目、しなやかな体つき
- 共通装備: 猫耳カチューシャ型の警護支援ユニットを常時装着する。支援ユニットはこの頭部装備だけで完結し、背面機械、外付けアーム、外骨格、追加ハーネスのような余計な機械装備は付けない。敵制圧に刃物は使わず、使うとしても非刃物の小型ツールに限る
- 共通性格: 忠実、冷静、毒舌気味。サトーの小さな幸福を本気で守っている
- 共通話し方: 短く要点だけ。独り言でだけ感情が少し漏れる
- 今回の欲望: サトーの休日を「少し上等な日曜日」として最後まで無傷で保存すること。今回はとくに、朝 / 昼 / 夜で変わる機嫌の良さを一つも欠けさせたくない
- 今回の恐れ: 休日導線が整っているぶん敵に読まれやすく、何でもないサービス動線がそのまま拉致導線に化けること
- 今回の服装・持ち物: 猫耳支援ユニットは常時装着。シリーズ正本のメイド基調シルエットを保った実務向け装いで動き、隠し通信具と非刃物の拘束解除ツールを携行する
- 今回の変化: 一日の終盤ほど「これだけ守っても本人は何も知らない」という可笑しさと愛着が強まり、夜の最後だけ少しだけ声に温度が滲む
- 画像プロンプト要約: 今回は canonical portrait をそのまま使用し、節画像やタイトル画像でも同じ顔立ち・髪色・猫耳装備を維持する。猫耳カチューシャ以外の追加メカ装備は付けず、ナイフや短剣などの刃物も持たせない
