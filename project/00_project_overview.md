# Project Overview

## Series Context
- シリーズ名: おっさんサトーの気ままなXX
- 今回の回で固定継承するもの: サトーとセラの基本設定、国家機密の裏設定、サトーは事件に気づかないという構図、3章6話12節の標準構成
- 今回の回で更新するもの: タイトル、サブタイトル、主題となる日常行為、季節、背景、各話の事件、各回専用の服装差分
- canonical portraits: project/assets/characters/sato.png / project/assets/characters/sera.png

## Logline
日曜日を丸ごと自分のために使おうと決めたサトーが、朝の珈琲、川沿いの散歩、昼のテラス、夕景の食事、夜のベランダという何気なくも華やかな休日を満喫する。その裏でセラは、同じ導線に何度も差し込まれる拉致工作を静かに潰し続ける。

## Audience
- 想定読者: 季節感のある日常描写と、気づかれない護衛アクションの落差を楽しみたい読者
- 読後感: サトーの穏やかな満足と、それが過剰に守られていた可笑しさが同時に残ること
- 連載ペース: 章ごとに確認しながら進める創読形式

## Creative Intent
- この回でやりたいこと:
  - 何も起きていないように見える一日が、実は一日中ぎりぎりで守られていたと読者だけが知る構図を徹底する
  - 朝 / 昼 / 夜で光と空気とサトーの満足の質を変え、日曜日を立体的に見せる
  - サトーの「丁寧に暮らす感じ」を、派手ではないが品のある休日として描く
- 絶対に入れたい要素:
  - 早朝の珈琲とベランダの静けさ
  - 白鷺川沿いの散歩とパンの匂い
  - 昼のテラス席と古本街
  - 夕景の食事と夜更けのベランダ
  - 猫耳メイド護衛が最後までサトーに気づかれないこと
- 避けたい要素:
  - 血生臭さや露骨な暴力
  - 国家機密の説明過多
  - サトーを鈍感ギャグにしすぎる描き方
  - セラを過剰に感情的に見せる描き方

## Product Intent
- 創読としてどんな体験にしたいか: サトーと一緒に気持ちよく題材を味わいながら、各話のセラパートで裏の事件の正体が見えてくる構成にする
- 画像生成の方針: canonical portraits を元に、今回はサトーだけ今回専用の sato-episode を生成し、セラはシリーズ正本の画像をそのまま使う。その後に背景・節画像・タイトル画像を作る
- レスポンシブ Web での見せ方: 各話2節の因果が縦スクロールで自然につながるように、サトー節は気分、セラ節は回収を明確に見せる。スマホ縦では横長絵がゆっくりパンし、横画面では節画像をフル表示する
