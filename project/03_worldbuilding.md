# Worldbuilding

## Setting Summary
現代日本によく似た世界。一般市民の日常と、サトーを守る国家規模の秘匿警護任務が同時進行している。今回は初夏の日曜日、東京側の生活圏で、朝のベランダから夜のベランダまでが一本の休日導線になる。

## Key Background Concepts
- 今回の背景 01: 早朝のマンション室内とベランダ。珈琲器具、鉢植え、柔らかい朝日、まだ人の少ない屋根並み
- 今回の背景 02: 白鷺川沿いの遊歩道とパン屋へ続く朝の導線。橋、川風、紙袋、日曜のまばらな人影
- 今回の背景 03: 白鷺川リバーフロント・コモンズの昼。テラス席、ガラスと木の外装、白い反射光
- 今回の背景 04: 川沿いの古本街。影のあるアーケード、古い木枠、紙の匂いがありそうな棚影
- 今回の背景 05: 夕景の川沿い食事処。水面の反射、橋の明かり、控えめな店灯り
- 今回の背景 06: 夜更けのリビングから続くベランダ。スタンドライト、レコード、甘味の皿、都心の小さな灯り

## Rules
- 魔法・技術: 超常要素はない。すべて現代的な諜報、拉致工作、事故偽装、通信妨害、拘束具で成立する
- 社会: 平和な日常の奥で、国家安全保障の最深部だけがサトーの扱いに神経を尖らせている
- 国家機密: 防衛の切り札となる極秘システムは、最終起動に登録済み本人の全身生体認証を必要とする
- 隠蔽構造: 登録事故は公表されておらず、政府内部は非常時だけ秘密裏にサトーを確保すればよいという危うい運用で問題を棚上げしている
- 国際情勢: 第三国はこの秘密を把握しており、サトーを生きたまま拉致して使用不能にすることを狙う
- 叙述方針: 国家機密は設定上の正本として保持し、本文では直接説明しすぎない
- 地理: 東京側生活圏。白鷺川と、その沿いの白鷺川リバーフロント・コモンズ、川沿い古本街、帰宅圏のマンション
- 季節: 初夏の晴れた日曜日。暑すぎず、風と光で気分の良さが立つ
- 建築: 川沿いの再開発商業街区、古い個人店アーケード、中層マンションのベランダ生活
- 禁則: 銃撃戦、爆発、群衆パニックのような大騒ぎは起こさない。すべては「休日の裏で静かに処理された」に収める

## Sensory Notes
- 色: 朝は薄金、昼は白と青、夕方は琥珀、夜は紺と暖色の室内灯
- 質感: リネン、紙袋、木の手触り、川風、テラスの金属椅子、ガラス越しの光
- 音: 電気ケトル、遠い川音、パン屋の扉、食器の触れ合い、ページをめくる音、レコードの針
- 匂い: 珈琲、焼きたてのパン、昼の川辺の乾いた風、古本の紙、夕食の湯気、夜の甘味
